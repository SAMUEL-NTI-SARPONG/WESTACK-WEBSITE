üåü Best Deployment Options

1. Netlify (Recommended) FREE EASIEST

‚úÖ Pros:
  • 100% Free for basic use
  • Extremely easy - just drag and drop
  • Automatic HTTPS (secure website)
  • Custom domain support
  • Fast global CDN
  • No technical knowledge needed
üìù Steps to Deploy:
  1. Go to www.netlify.com
  2. Click "Sign up" (use email or GitHub)
  3. After logging in, click "Add new site" ‚Üí "Deploy manually"
  4. Drag the entire westack-website folder into the box
  5. Wait 30 seconds - Your site is LIVE!
  6. Netlify gives you a URL like: your-site-name.netlify.app
  7. Optional: Add your custom domain (westackengineering.com)
üí° Pro Tip: You can update your website anytime by dragging the folder again!

2. GitHub Pages FREE EASY

‚úÖ Pros:
  • 100% Free forever
  • Integrated with Git (version control)
  • Good for developers
  • Automatic HTTPS
  • Custom domain support
‚ùå Cons:
  • Requires basic Git knowledge
  • Slightly more technical
üìù Steps to Deploy:
  1. Create account at github.com
  2. Install GitHub Desktop
  3. Create new repository called "westack-website"
  4. Add all your website files to the repository
  5. Commit and push to GitHub
  6. Go to repository Settings ‚Üí Pages
  7. Select branch "main" and folder "/" (root)
  8. Click Save - site goes live at yourusername.github.io/westack-website

3. Traditional Web Hosting (cPanel) PAID

‚úÖ Pros:
  • Full control over domain and email
  • Can host multiple websites
  • Professional email addresses (info@westack.com)
  • Good for business websites
‚ùå Cons:
  • Costs $3-10/month
  • Requires domain purchase (~$10-15/year)
üìù Steps to Deploy:
  1. Buy hosting from: Namecheap, Bluehost, SiteGround, or local Ghana providers
  2. Register domain: westackengineering.com or westack.com.gh
  3. Log into cPanel (hosting control panel)
  4. Open "File Manager"
  5. Navigate to public_html folder
  6. Upload all files from westack-website folder
  7. Make sure index.html is in the root
  8. Visit your domain - Website is live!

üåê Custom Domain Setup

Want westackengineering.com instead of netlify.app?
  1. Buy domain from Namecheap, GoDaddy, or Register.com.gh
  2. In your hosting platform (Netlify/Vercel), go to Domain Settings
  3. Add your custom domain
  4. Update DNS records at your domain registrar (they provide instructions)
  5. Wait 24-48 hours for DNS to propagate
  6. Your site is now at your custom domain!

üìß Setting Up Contact Form

Your contact form needs a backend to send emails. Options:
  1. Formspree (Easiest):
    • Sign up at formspree.io (free plan available)
    • Get your form endpoint
    • Update form action in contact.html
  2. Netlify Forms (if using Netlify):
    • Just add netlify attribute to your form
    • Forms automatically work!
  3. Email.js:
    • Free service for sending emails from contact forms
    • Easy JavaScript integration
