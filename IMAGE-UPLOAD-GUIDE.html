üîß Step-by-Step Image Upload Process

Step 1: Prepare Your Images
  • Resize images to recommended dimensions
  • Compress images for web (use tools like TinyPNG.com)
  • Rename files to match the required names above
  • Keep file sizes under 500KB for faster loading
Step 2: Upload to img/ Folder
  • Navigate to: westack-website/img/
  • Copy your prepared images into this folder
  • Replace existing placeholder images
  • Keep the same file names or update HTML files accordingly
Step 3: Test Locally
  • Open index.html in your web browser
  • Navigate through all pages
  • Check that all images load correctly
  • Verify images look good on mobile (resize browser window)
Step 4: Deploy to Web Hosting
  • Upload the entire westack-website folder to your hosting
  • Or use GitHub Pages, Netlify, or Vercel (see README.md)
  • Test the live website
  • Share with colleagues for feedback

üí° Pro Tips

  • Image Optimization: Use tools like TinyPNG or ImageOptim to reduce file sizes without losing quality
  • Consistent Style: Try to maintain consistent lighting, colors, and composition across project photos
  • High Quality: Use high-resolution photos that will look sharp on large screens
  • Mobile Testing: Always check how images look on mobile devices
  • Alternative Text: Consider adding descriptive alt text for accessibility
